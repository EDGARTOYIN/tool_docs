toolpocket_app.views.add_to_favorites(request)[fuente]

Vista para agregar un producto a favoritos.

Verifica si el usuario está autenticado y si la solicitud es de tipo POST. Obtiene el ID del producto de la solicitud POST y busca el producto correspondiente en la base de datos. Obtiene el perfil de usuario asociado al usuario autenticado y agrega el producto a la lista de productos favoritos del perfil. Muestra un mensaje de éxito y redirige a la página de inicio.

Returns:

HttpResponse: El objeto de respuesta HTTP que redirige a la página de inicio.

toolpocket_app.views.collectionsview(request, slug)[fuente]

Vista para la página de vista de colecciones.

Verifica si existe una categoría con el slug dado y estado “0”. Si existe, filtra los productos asociados a dicha categoría y crea un contexto para renderizar la plantilla “toolpocket_app/products/index.html” con los productos y la categoría obtenidos. Si no existe, muestra un mensaje de advertencia y redirige a la página de colecciones.

toolpocket_app.views.productview(request, cate_slug, prod_slug)[fuente]

Vista para la página de vista de productos.

Verifica si existe una categoría y un producto con los slugs dados y estados “0”. Si existen, filtra el producto asociado al slug de producto dado y crea un contexto para renderizar la plantilla “toolpocket_app/products/view.html” con el producto obtenido. Si no existen, muestra un mensaje de error y redirige a la página de colecciones.

HttpResponse: El objeto de respuesta HTTP que renderiza la página de vista de productos.
